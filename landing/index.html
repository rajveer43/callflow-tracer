Architecture

How it works.

1
Your Python Code
sys.setprofile hooks every function call with nanosecond precision.
2
CallGraph + LLM Patches
Concurrent: runtime call edges build in memory while SDK patches intercept Anthropic / OpenAI calls.
3
Merge ✦
Static AST + runtime graph + LLM spans fuse into one UnifiedGraph.
4
Render Everywhere
Flame graph HTML, Neo4j Cypher, Chrome trace JSON, or live WebSocket dashboard.
